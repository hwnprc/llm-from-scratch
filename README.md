# llm-from-scratch

## 학습 기록

### 2026-07-01: C언어 Hello World 프로그램

#### 작업 내용
1. 기본 Hello World 프로그램 작성 ([hello.c](hello.c))
2. 콘솔 출력에서 파일 출력으로 수정

#### 시행착오
1. **첫 번째 시도**: 콘솔에 출력하는 기본 프로그램
   - `printf()`를 사용하여 터미널에 출력

2. **사용자 요청**: 출력을 파일에 저장하도록 변경

3. **최종 버전**: 파일 입출력 사용
   - `fopen()`으로 파일 열기
   - `fprintf()`로 파일에 쓰기
   - `fclose()`로 파일 닫기
   - 에러 처리 추가 (파일 열기 실패 시)
   - 결과: [output.txt](output.txt)에 "Hello, World!" 저장

#### 배운 점
- C언어의 파일 입출력 함수 사용법
- `FILE` 포인터와 파일 스트림 관리
- 에러 처리의 중요성
